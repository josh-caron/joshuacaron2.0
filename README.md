# Josh Caron's Portfolio

Welcome to my personal portfolio website. This site showcases my background as a second-year Computer Science student at the University of Florida, my academic achievements, and my professional experience.


## Built With

- **[Astro](https://astro.build/)** - Static site generator for modern web apps
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[Tabler Icons](https://tabler.io/icons)** - Free and open source icons
- **TypeScript** - For type-safe configuration

## Local Development

To run this portfolio locally:

```bash
git clone https://github.com/josh-caron/portfolio.git
cd portfolio
npm install
npm run dev
```
